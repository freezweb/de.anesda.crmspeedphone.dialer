# Store-Eintrag (Deutsch)

## Name

SpeedPhone Dialer

## Kurzbeschreibung / Untertitel

CRM-Anrufe sicher auf dem Handy starten

## Beschreibung

SpeedPhone Dialer verbindet Ihr Smartphone mit CRM SpeedPhone. Öffnen Sie im CRM „Handy koppeln“, scannen Sie den kurzlebigen QR-Code und starten Sie anschließend die im CRM hinterlegte Telefonnummer mit einem Klick auf Ihrem Smartphone.

Die Nummer muss nicht mehr abgetippt oder über Messenger übertragen werden. Der zugehörige CRM-Datensatz bleibt während des Gesprächs im Browser geöffnet, damit Ergebnis, Notiz und Wiedervorlage direkt dokumentiert werden können.

Funktionen:

- einfache Kopplung per QR-Code ohne Eingabe von Serveradresse oder CRM-Passwort
- benutzerbezogene Gerätezuordnung
- kurzlebige Anrufaufträge
- sicher im Betriebssystem gespeichertes Gerätetoken
- direkter Anrufstart auf Android nach erteilter Telefonberechtigung
- systemeigene Anrufbestätigung auf iPhone
- keine Werbung und kein Tracking

Die App benötigt eine erreichbare SuiteCRM-Installation mit CRM SpeedPhone ab Version 1.4.0. Sie ist kein eigenständiges CRM und zeichnet keine Gespräche auf.
